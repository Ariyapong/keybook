# keybook v0.2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship v0.2 — readline-style search-input editing (Phase A: `⌥⌫`, `⌃W`, `⌃U`) and a Homebrew tap for `brew install ariyapong/tap/keybook`.

**Architecture:** Two independent feature areas. **(1)** A small TUI change in `src/tui/` adding a pure `deleteWordBack` helper, three new bindings in `App.tsx`, and a footer hint update. **(2)** A sibling GitHub repo `Ariyapong/homebrew-tap` containing a Ruby formula that downloads the published npm tarball. No schema, loader, search, or CLI changes.

**Tech Stack:** TypeScript + Ink (TUI feature, no new deps). Homebrew Ruby formula sourced from the npm tarball (external repo, no Node dep in the main repo).

**Reference spec:** `docs/superpowers/specs/2026-06-02-keybook-v0.2-design.md`

**Conventions:**
- TDD: write the failing test, watch it fail, implement, watch it pass.
- One commit per task. All `git`/`gh`/`pnpm` commands run from the repo root.
- Match v0.1 commit-style: `feat(tui): …`, `chore(release): …`, etc.
- The Homebrew tap lives in a sibling repo, **not** under `keybook/`. Commands prefixed with `cd ../homebrew-tap` operate in that repo.
- After each in-repo task, `pnpm test`, `pnpm typecheck`, and `pnpm lint` must stay green.

```ts
/**
 * Drop the run of trailing whitespace, then the run of trailing non-whitespace.
 * Mirrors readline's `unix-word-rubout` / `backward-kill-word` behavior so a
 * single press eats both the spaces and the word immediately before them.
 */
export function deleteWordBack(query: string): string {
  const trimmed = query.replace(/\s+$/, "");
  const lastSpace = trimmed.lastIndexOf(" ");
  if (lastSpace === -1) return "";
  return trimmed.slice(0, lastSpace + 1);
}
```

```ts
    const { lastFrame, stdin } = render(<App entries={entries} />);
    await tick();

    await tick();
    // Ink reports Option+Backspace as Escape-prefixed Delete (0x7f).

    expect(lastFrame()).not.toContain("clear new");
  });

  it("deletes the previous word on Ctrl+W", async () => {
    const { lastFrame, stdin } = render(<App entries={entries} />);
    await tick();
    stdin.write("clear new");
    await tick();
    stdin.write("\x17"); // Ctrl+W
    await tick();
    expect(lastFrame()).toContain("search: clear ");
    expect(lastFrame()).not.toContain("clear new");
  });

  it("clears the entire query on Ctrl+U", async () => {
    const { lastFrame, stdin } = render(<App entries={entries} />);
    await tick();
    stdin.write("anything goes");
    await tick();
    stdin.write("\x15"); // Ctrl+U
    await tick();
    expect(lastFrame()).toContain("search: ");
    expect(lastFrame()).not.toContain("anything");

    const { lastFrame, stdin } = render(<App entries={entries} />);
    await tick();
    stdin.write("clear");
    await tick();
    stdin.write("\x7f"); // Backspace
    await tick();
```
